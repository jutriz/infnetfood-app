# infnetfood-app
Aplicativo mobile desenvolvido em React Native com Expo para pedidos de comida, incluindo login, listagem de categorias, carrinho de compras, checkout com cálculo de total, visualização de pedidos e suporte a tema claro/escuro.

## Acessar o projeto online
https://snack.expo.dev/@julia_chuery/infnetfood-at-desenvolvimento-mobile-com-react-native

## Funcionalidades
- Login com validação (email: Julia@gamil.com e senha: julia123)
- Listagem de categorias
- Produtos com imagens e preços
- Carrinho de compras
- Checkout com cálculo de total
- Visualização de pedidos
- Tema claro e escuro

## Observação
O projeto foi desenvolvido utilizando Expo Snack, uma plataforma online que permite executar aplicações React Native diretamente no navegador, sem necessidade de instalação local. :contentReference[oaicite:0]{index=0}
